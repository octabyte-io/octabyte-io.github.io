## FAQ

- **Can I access my server and make customizations?**
  Yes, you can access your server and customize it as needed. However, this option is not available in the Startup plan. For other plans, we provide an SSH key on request, allowing you to securely access your server.

- **Is my server dedicated only to my app?**
  Yes, your server is dedicated to your app. However, in the Startup plan, the server is shared, meaning its resources are also used by neighboring apps.

- **Can I install more than one app on a single server?**
  Yes, you can install up to 4 apps on the Business plan and up to 10 apps on the Enterprise plan. In the Startup and Professional plans, only one app can be installed per server.

- **Can I upgrade my server later?**
  Yes, you can upgrade your server at any time without any data loss.

- **Can I get custom server specifications?**
  Yes, we offer customizable server specifications tailored to your requirements. Simply contact our support team and share your needs with them.
